## General info
This group project is a pro-healthy app created during frontend bootcamp (not finished).
	
## Features
User can:
* sign in/sign up, remind password
* take up the challenges 
* search challenges
* add notes to mood diary
* remove notes from mood diary
* share app on facebook

## Technologies
Project is created with:
* React.js
* Redux
* Firebase
* Material-UI
	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
